# Go Beauty Bangladesh - Frontend

A modern React-based e-commerce platform for beauty, skincare, and cosmetics products tailored for the Bangladeshi market.

## Overview
Go Beauty Bangladesh (gobeauty.bd) is a full-featured beauty products marketplace offering skincare, makeup, and hair & body care essentials. The frontend is built with React and Vite for optimal performance and developer experience.

## Features
- 📦 Comprehensive product catalog with 3 main categories and multiple subcategories
- 🛍️ Advanced product filtering by skin type and category
- ⭐ Product ratings, reviews, and customer feedback
- 🏷️ Dynamic pricing with discount tracking
- 🎨 Product image galleries
- 📱 Responsive design optimized for Bangladesh market
- ⚡ Fast loading with Vite build tool
- 🔄 Real-time stock availability

## Tech Stack
- **Frontend Framework:** React 19.2.3
- **Build Tool:** Vite 8.0.16
- **React Plugin:** @vitejs/plugin-react 5.1.1
- **Runtime:** Node.js (ES Modules)

## Quick Start
```bash
npm install      # Install dependencies
npm run dev      # Start development server (http://127.0.0.1:5173)
npm run build    # Build for production
